JavaScript y TypeScript en Español

Instrucciones de Instalación

  • Descomprime el archivo ZIP.
  • Coloca todo el contenido en la carpeta de tu proyecto.
  • Agrega la siguiente línea en tu código:
    • TypeScript: import * as espanol from './importar.ts';
    • JavaScript: import * as espanol from './importar.js';

Uso

Para utilizar las funcionalidades del proyecto, escribe:

espanol.*(el comando que quieres utilizar)
