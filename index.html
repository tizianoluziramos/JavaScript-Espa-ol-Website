JavaScript and TypeScript in English

Installation Instructions

  • Unzip the ZIP file.
  • Place all content in your project folder.
  • Add the following line to your code:
    • TypeScript: import * as english from './import.ts';
    • JavaScript: import * as english from './import.js';

Usage

To use the project's functionalities, write:

english.*(the command you want to use)
