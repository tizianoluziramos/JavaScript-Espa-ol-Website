JavaScript Español

Últimas Actualizaciones y Changelog

Bienvenidos a la documentación de cambios del proyecto JavaScript Español, una iniciativa para llevar a la comunidad de habla hispana herramientas y módulos en JavaScript en español.

v1.1.3 - PreRelease

Changelog:

  • Eliminación de console.log en la función esAnagrama para mejorar eficiencia.
  • Finalización de procesos con process.exit() al terminar código.
  • Creación de carpetas F1 y Update, y eliminación de carpeta vacía BSOD.
  • Añadido creador de redes neuronales y funciones API para información de Franco Colapinto y Max Verstappen.

v1.1.2 - Última Release

Correcciones:

  • Eliminación de console.log en esAnagrama.
  • Nuevo archivo autoFixCode.ts (vacío, por ahora).
  • Corrección en ejecución de código con TypeScript.

v1.1.0 - Funcionalidades Nuevas

Añadidos:

  • Funciones para crear bots de Discord y Slack fácilmente.
  • Compatibilidad con Node.js y funcionalidades SFTP.
  • Nuevas funciones de consola y herramientas de validación.
